CPU Scheduling

Benjamin Rosales

The CPU Scheduling Simulator is a software that simulates and visualizes the different techniques of CPU Scheduling, which includes First Come, First Serve (FCFS), Shortest Job First (SJF), Shortest Remaining Job First (SRJF), Round Robin (RR), Non – Preemptive (NP) and Preemptive. This program calculates the Waiting time, Response time and Turnaround time, as well as its averages. A Gantt Chart representation of the scheduling is also displayed, with the processes represented by different colors to distinguish easier. A comparative table is accessible within the program for the comparison of the said techniques.
