Arduino RGB
Wendell Herrero

In this setup, the materials used are RGB LED, potentiometer, photoresistor, Arduino UNO, resistors, and jumper wires. It lights up when it is dark around. It makes it happen because of photoresistor which reads the light intensity around. And with potentiometer, it can change the color of the LED. The LED used here is not a simple LED because it is an RGB. It combines the primary color of Red, Green and Blue to create different colors.
